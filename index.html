Conclusiones

Si hay etiquetas que no aparecen en los primeros bloques de codigo, pero se agregan despues no se muestran cuando se corre el servidor, por ejemplo en el ejercicio de crear una ficha, La primer ficha no tenia las etiquetas de AFP, ISSS y DUI pero las fichas 2 y 3 si las tenian pero cuando inicie el servidor para mostrar la pagina no se mostraba el texto de ISSS; AFP y DUI.

Al agregar el estilo con CSS la pagina se muestra sin las etiquetas, muy parecido a HTML

La coversion de xml a JSON es muy sencilla si se usa un convertidor

Agregar CSS a xml es muy parecido (por no decir igual) a agregarlo a HTML

Un documento XML puede estar bien formado, pero aun asi necesita un DTD para que sea valido
